# libsheaf Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the sheaf cohomology consistency check library — computes the coboundary operator δ over a local neighborhood to detect logical contradictions, and generates HTML comment annotations for the agent.

**Architecture:** libsheaf is decoupled from MemoryGraph. Instead of importing the full graph, `CoboundaryOperator::compute_local()` takes a lightweight `Neighborhood` struct (the new node's embedding + a list of neighbor embeddings with edge types). The engine scheduler is responsible for building the Neighborhood from the MemoryGraph before calling libsheaf. This keeps libsheaf independently testable with no dependency on the engine module. libsheaf depends on libmetric only for the `GaussianNode` type (used for embeddings).

**Tech Stack:** C++23, Google Test, depends on libmetric (GaussianNode)

#pragma once

#include <cstdint>
#include <span>
#include <string>
#include <vector>

namespace slm::sheaf {

/// Edge type in the local knowledge graph.
enum class EdgeType : uint8_t {
    Structural = 0,  // parent-child or sibling (from scene graph)
    KNN = 1,         // cross-branch k-nearest neighbor
};

/// An edge from the new node to a neighbor in the local neighborhood.
struct Edge {
    uint32_t neighbor_index;          // index into Neighborhood::neighbor_mus
    EdgeType type;
    std::vector<float> relation;      // expected semantic difference r_ij
};

/// Lightweight input to the coboundary operator.
/// Built by the engine scheduler from the MemoryGraph before calling libsheaf.
/// This decouples libsheaf from the MemoryGraph implementation.
struct Neighborhood {
    /// Embedding of the newly ingested node.
    std::span<const float> new_node_mu;
    /// Text of the newly ingested node (for annotation).
    std::string new_node_text;

    /// Embeddings of neighbor nodes.
    std::vector<std::vector<float>> neighbor_mus;
    /// Texts of neighbor nodes (for annotation).
    std::vector<std::string> neighbor_texts;

    /// Edges from the new node to its neighbors.
    std::vector<Edge> edges;
};

} // namespace slm::sheaf
```

#pragma once

#include <cstdint>
#include <vector>
#include <sheaf/neighborhood.hpp>

namespace slm::sheaf {

/// Result of the local coboundary computation.
struct CoboundaryResult {
    /// L2 norm of the coboundary across all edges (weighted by edge type).
    float norm{0.0f};
    /// Indices into Neighborhood::neighbor_mus of neighbors that individually
    /// exceeded the contradiction threshold.
    std::vector<uint32_t> conflicting;
};

/// Computes the coboundary operator δ over a local neighborhood.
///
/// For each edge (new_node, neighbor_i) with relation r_i:
///   δ_i = mu_new - mu_neighbor_i - r_i
///
/// Structural edges are weighted 2x higher than KNN edges in the norm.
/// Individual neighbors with ||δ_i|| > threshold are flagged as conflicting.
class CoboundaryOperator {
public:
    /// Compute δ over the local neighborhood of a newly ingested node.
    /// @param hood       The local neighborhood (new node + neighbors + edges)
    /// @param threshold  Per-edge contradiction threshold
    /// @return           Aggregate norm and list of conflicting neighbor indices
    CoboundaryResult compute_local(
        const Neighborhood& hood,
        float threshold
    ) const;
};

} // namespace slm::sheaf
```

```cpp
#include <sheaf/coboundary.hpp>
#include <cmath>
#include <cassert>

namespace slm::sheaf {

CoboundaryResult CoboundaryOperator::compute_local(
    const Neighborhood& hood,
    float threshold
) const {
    CoboundaryResult result;

    if (hood.edges.empty()) {
        return result;
    }

    float total_weighted_sq = 0.0f;

    for (const auto& edge : hood.edges) {
        assert(edge.neighbor_index < hood.neighbor_mus.size());
        const auto& mu_neighbor = hood.neighbor_mus[edge.neighbor_index];
        const auto& relation = edge.relation;

        uint32_t dim = static_cast<uint32_t>(hood.new_node_mu.size());
        assert(mu_neighbor.size() == dim);
        assert(relation.size() == dim);

        // Compute δ_i = mu_new - mu_neighbor - relation
        float edge_sq_norm = 0.0f;
        for (uint32_t d = 0; d < dim; ++d) {
            float delta = hood.new_node_mu[d] - mu_neighbor[d] - relation[d];
            edge_sq_norm += delta * delta;
        }

        float edge_norm = std::sqrt(edge_sq_norm);

        // Weight: structural edges count 2x
        float weight = (edge.type == EdgeType::Structural) ? 2.0f : 1.0f;
        total_weighted_sq += weight * edge_sq_norm;

        // Flag individual contradictions
        if (edge_norm > threshold) {
            result.conflicting.push_back(edge.neighbor_index);
        }
    }

    result.norm = std::sqrt(total_weighted_sq);
    return result;
}

} // namespace slm::sheaf
```

#include <sheaf/annotation.hpp>
#include <sstream>
#include <iomanip>

namespace slm::sheaf {

namespace {

/// Escape double quotes in text for safe embedding in the annotation.
std::string escape_quotes(const std::string& text) {
    std::string result;
    result.reserve(text.size());
    for (char c : text) {
        if (c == '"') {
            result += "\\\"";
        } else {
            result += c;
        }
    }
    return result;
}

} // namespace

std::string format_annotation(const Annotation& ann) {
    std::ostringstream oss;
    oss << std::fixed << std::setprecision(2);
    oss << "<!-- cohomology: superseded \""
        << escape_quotes(ann.superseded_text)
        << "\" by \""
        << escape_quotes(ann.superseding_text)
        << "\", δ-norm=" << ann.delta_norm
        << " -->";
    return oss.str();
}

} // namespace slm::sheaf

    EXPECT_EQ(result.conflicting[0], 0u);

    // Step 2: Generate annotation for each conflicting neighbor
    for (auto neighbor_idx : result.conflicting) {
        Annotation ann;
        ann.superseded_text = hood.neighbor_texts[neighbor_idx];
        ann.superseding_text = hood.new_node_text;
        ann.delta_norm = result.norm;

        std::string annotation = format_annotation(ann);

        // Verify annotation format
        EXPECT_NE(annotation.find("proxy IP is 10.0.0.1"), std::string::npos);
        EXPECT_NE(annotation.find("proxy IP is 10.0.0.5"), std::string::npos);
        EXPECT_NE(annotation.find("δ-norm="), std::string::npos);
        EXPECT_EQ(annotation.substr(0, 5), "<!-- ");
        EXPECT_EQ(annotation.substr(annotation.size() - 4), " -->");
    }
}

TEST(SheafIntegration, NoContradictionNoAnnotation) {
    // Consistent update: adding a new fact that doesn't contradict anything
    std::vector<float> mu_new = {1.0f, 0.0f, 0.0f};
    std::vector<float> mu_old = {0.0f, 1.0f, 0.0f};
    // Relation captures the expected difference
    std::vector<float> rel = {1.0f, -1.0f, 0.0f};

    Neighborhood hood;
    hood.new_node_mu = mu_new;
    hood.new_node_text = "new fact";
    hood.neighbor_mus = {std::vector<float>(mu_old.begin(), mu_old.end())};
    hood.neighbor_texts = {"old fact"};
    hood.edges = {{0, EdgeType::Structural, rel}};

    CoboundaryOperator op;
    auto result = op.compute_local(hood, 0.5f);

    EXPECT_NEAR(result.norm, 0.0f, 1e-5f);
    EXPECT_TRUE(result.conflicting.empty())
        << "Consistent nodes should produce no contradictions";
}

## Summary

After completing all 4 tasks, libsheaf provides:

| Component | What it does |
|---|---|
| `Edge` / `EdgeType` | Edge types (Structural vs KNN) with relation vectors |
| `Neighborhood` | Lightweight input struct decoupled from MemoryGraph |
| `CoboundaryOperator::compute_local()` | Computes δ over local 1-ring, structural edges weighted 2x |
| `CoboundaryResult` | Aggregate norm + list of conflicting neighbor indices |
| `Annotation` | Struct holding superseded/superseding text + δ-norm |
| `format_annotation()` | Generates `<!-- cohomology: ... -->` HTML comment |

libsheaf depends on libmetric (for GaussianNode type) but is decoupled from MemoryGraph. The engine scheduler builds a `Neighborhood` from the graph before calling `compute_local()`.
